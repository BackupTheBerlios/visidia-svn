VISIDIA: VIsualisation and SImulation of DIstributed Algorithms.

1.) les repertoires
    +  VISIDIA_HOME/classes : repertoire de destination de la
    compilation des sources java, contient les ressources (images,
    graphe).

    + VISIDIA_HOME/sources : repertoire des sources de visidia.
    
    + VISIDIA_HOME/applet : contient la forme applet de visidia.

2.) environnement Java:
    Visidia a ete developpe et teste avec jdk1.3. les scripts ont ete
    ecrit pour jdk1.3.
4.) les scripts
    + compile.sh (.bat): compilation des sources de visidia.
    + visidia.sh (.bat): lancement de visidia.
    + createjar.sh (.bat): creation de l'archive jar de visidia dans
    le repertoire classes.
