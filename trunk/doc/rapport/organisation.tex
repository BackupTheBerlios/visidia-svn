\section{Les moyens techniques}

Afin de mener à bien notre projet, nous avons utilisés un portail évolué de
développement fournissant plusieurs outils : le portail berliOs.de \footnote{\url{http://www.berlios.de}}
\\ \newline
Pour améliorer la communication avec notre client et notre responsable
pédagogique, nous avons mis en place un wiki sur lequel on pouvait notamment
consulter les compte rendu de réunion.\\

Nous avons déployés plusieurs listes de diffusion pour faciliter la communication :
\begin{itemize}
  \item Une liste pour tous les développeurs
  \item Une liste de diffusion pour communiquer avec le client
  \item Une liste de diffusion pour communiquer avec le responsable pédagogique \\
\end{itemize}

Nous avons utilisé également un gestionnaire de versions, Subversion
, qui nous permet de maintenir les sources de \visidia. 
Les clients ont également la possibilité de
suivre l'avancement du projet et d'accéder aux sources, grâce à deux
interfaces viewcvs  et websvn .  Il est
également possible de télécharger la dernière version des sources du
projet:
\begin{verbatim}
svn checkout svn://svn.berlios.de/visidia/trunk
\end{verbatim}
