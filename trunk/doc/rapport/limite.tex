
\paragraph{Lisibilité et extensibilité}

Cette  partie ne concerne  pas directement  la simulation  des agents,
mais \visidia dans  son ensemble : le programme souffre  de son âge et
du fait d'être passé entre de nombreuses mains d'élèves ingénieurs peu
expérimentés (y compris nous).

De ce fait, il résulte  un programme qui fonctionne correctement, mais
aux possibilités  d'extension faibles (peu  d'utilisation du mécanisme
d'héritage). Ainsi,  les rajouts se greffent directement  dans le code
existant,  ou  via  des   copier-coller,  augmentant  les  risques  de
plantages et/ou d'incohérences.

Les parties sur  lesquelles nous avons travaillé accusant  ce genre de
problème sont  l'interface graphique principalement,  et le simulateur
dans une moindre mesure.
