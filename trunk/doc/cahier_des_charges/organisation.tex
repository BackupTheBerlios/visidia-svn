L'équipe  des réalisateurs rencontrera  les clients  une fois  par semaine
pour faire le point sur le travail effectué. L'horaire du vendredi matin à
10h  a été retenu  pour cette  entrevue hebdomadaire.  Les clients  et les
réalisateurs resterons  en contact et  pourrons échanger sur le  projet au
moyen d'une liste de diffusion \ref{mailing-list}.

\section{Échéancier}

vendredi 14 avril, 17h00 dernière limite.

La remise du projet donnera lieu à une soutenance dont la date est
fixée au jeudi 27 avril.

\subsection{Utilisation d'une plate-forme de développement}

Le projet sera l'occasion d'utiliser un portail évolué de
développement sur le modèle du célèbre portail
%\href{http://sourceforge.net}{SourceForge}.
SourceForge \cite{SourceForge}.
Notre choix s'est porté sur le portail
%\href{http://www.berlios.de}{\berlios} 
\berlios \cite{BerliOs}  pour des raisons de rapidité  des serveurs. Notre
projet \visidia dispose ainsi d'une page d'accueil \cite{visidiaPFA}:
%\href{http://visidia.berlios.de}{\visidia.\berlios} 
\begin{verbatim}
http://visidia.berlios.de
\end{verbatim}

Cette page  d'accueil est un  wiki \cite{wiki}. Cela permet  une évolution
plus souple des informations présentées. Cette page est le point de départ
vers  les  différents services  fournis  par  \berlios.  Tous les  modules
décrits par la suite peuvent être  retrouvés à partir de la page d'accueil
du projet.\\

Tout d'abord les sources de notre extension de \visidia sont
maintenues grâce au gestionnaire de version Subversion
\cite{subversion}. Le client a la possibilité de suivre l'avancée du
projet en accédant aux sources. Deux interfaces web permettent l'accès
aux sources: 
%\href{http://svn.berlios.de/viewcvs/visidia/}{viewcvs}
viewcvs \cite{viewcvs} 
et
%\href{http://svn.berlios.de/wsvn/visidia/}{websvn}
websvn \cite{websvn}. Il est également possible de télécharger anonymement
les sources depuis le dépôt Subversion:
\begin{verbatim}
svn checkout svn://svn.berlios.de/visidia/trunk
\end{verbatim}

L'équipe  a aussi mis  en place  des mailing-lists  gérées par  le portail
\berlios.   Une  mailing-list  permet  aux  clients de  joindre  tous  les
réalisateurs  et vice  versa. L'adresse  de cette  liste de  diffusion est
\label{mailing-list}:
\begin{verbatim}
visidia-pfa@berlios.de
\end{verbatim}

Il est possible  de consulter les archives de cette  liste de diffusion au
moyen d'une interface web \cite{mailing-list}.
